My attempt in the world of asynchronicity

### Initialize

```
git clone https://github.com/wootsaejao/just-another-online-store.git
cd just-another-online-store
```

### Development

```
npm run test:watch
npm run dev
```

### Production

```
npm run build
npm run start
```
